# Reference Check

Keine Behauptung ohne Beleg.

## Beispiel

**Prompt:**
```text
Beantworte die Frage basierend auf dem Text.
Füge hinter jedem Satz eine Referenz ein, z.B. [Seite 2, Zeile 10].
Sätze ohne Referenz sind verboten.
```
## Strategie

Der Goldstandard für RAG-Systeme.
