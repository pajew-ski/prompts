# Token Budget

Modelle labern gerne. Setze harte Grenzen.

## Beispiel

**Prompt:**
```text
Erkläre die Relativitätstheorie.
Token Budget: Maximal 50 Wörter.
Jedes Wort darüber wird bestraft.
```
## Strategie

Zwingt das Modell, sich auf das Wesentliche zu fokussieren ("Dichte").
